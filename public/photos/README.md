# Apartment photo assets

Place the apartment photos in this folder with these exact names:

- `01-laundry.jpg`
- `02-kitchen-wide.jpg`
- `03-fridge.jpg`
- `04-bathroom.jpg`
- `05-range.jpg`
- `06-open-kitchen.jpg`

The landing page uses `06-open-kitchen.jpg` as the hero image and all six files in
the gallery. If a file is missing, the app shows a labeled fallback tile until the
real photo is added.
