# Fitchburg Main Street Landing

Vite landing page for 633-639 Main Street Apartments in Fitchburg, MA.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Photos

Add the original photos to `public/photos/source` using the names listed in
`public/photos/README.md`, then convert them:

```bash
npm run photos:webp
```
